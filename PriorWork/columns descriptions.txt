Transaction Report
Description: This describes all the transactions that are done by the customers in any way other than using POS machines.

No	Columns	Description
1	DESCRIPTION	The name of the bank’s product the customer uses to make the transaction.

2	'RIMNO'	Customer ID 
3	'POST DATE'	Date that the transaction is officially recorded in the bank’s system.
4	'TRXN DATE'	When the transaction exactly occurred
5	'CCY'	Currency code (ISO 4217) (ex. 840 USD, 818 EGP)
6	'ORIG AMOUNT'	Original Amount in original currency
7	'EMBEDDED _FEE'	Currency exchange fee
8	'BILLING AMT'	Total transaction amount in EGP
9	'MCC'	Merchant type, 
10	'MERCHNAME'	Merchant data
11	'MERCH ID'	
12	'SOURCES'	
13	'SETTLEMENT AMT'	Total amount in USD if currency is not EGP
And in EGP if it was originally in EGP
14	'SETTLEMENT CCY'	818 if originally in EGP, otherwise 840 (USD)
15	'BANKBRANCH'	Name of the branch the customer opened their account from
16	'TRXN COUNTRY'	Country the transaction happened in 
17	'REVERSAL FLAG'	If the transaction is reversed
